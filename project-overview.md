# Project Overview

Project Name: Hello World Box

Description: A single-page website that displays "Hello World" inside an HTML `<div>` box.

Features:

*   Simple HTML structure.
*   "Hello World" text within a `<div>` element.

Target Audience: Developers testing the GenStack platform, or anyone needing the most basic webpage possible.

Technology Stack:

*   Cloudflare next-on-pages framework (NextJS) - but really just using it for its deployment capabilities, since the code is pure HTML.
